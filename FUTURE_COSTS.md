# Stuttee (S.T) - 향후 추가 비용 항목 (초안)

현재 모크 기반 MVP 이후 확장 시 발생할 수 있는 잠재 비용을 정리한 메모입니다. 금액은 사용량/지역/프로바이더에 따라 달라질 수 있습니다.

## 1) AI API (OpenAI 연동 시)
- OpenAI API 사용료(1K 토큰당): 모델(GPT-4/4o/3.5 등)에 따라 다름. 1회 요청 비용 = 프롬프트 토큰 + 응답 토큰 합산 기준.
- 프롬프트 최적화로 비용 절감: 문제 수/난이도/유형 제한, 해설 길이 제한, 컨텍스트 최소화.
- 에러/재시도 정책: 재시도 횟수 제한, 서버 측 max tokens 설정으로 과금 폭주 방지.

## 2) 호스팅
- 백엔드: 소형 VM/컨테이너(예: 1 vCPU/1GB) 또는 서버리스 함수. 상시 구동 저사양 VM은 월 약 $5–$15 수준, 서버리스는 호출 횟수에 따라 변동.
- 프런트엔드: 정적 호스팅(Netlify/Vercel/S3+CDN 등). 초기 트래픽은 무료/저가 구간이 많음.
- TLS/SSL: 매니지드 호스팅은 보통 포함(LetsEncrypt). 자가 호스팅 시 인증서 설정 시간/비용 고려.

## 3) 도메인 & CDN
- 도메인: 연 약 $10–$20 수준이 일반적.
- CDN(정적 자산/엣지 API 캐시): 보통 번들 또는 GB당 과금. 초기 트래픽은 낮은 편.

## 4) 스토리지/데이터베이스(향후 추가 시)
- 매니지드 DB(Postgres/MySQL 등) 엔트리 티어: 월 약 $10–$30. 계정/히스토리 등 영구 저장이 필요해질 때 도입.
- 오브젝트 스토리지(내보내기/로그): GB당 소액, 다만 외부 전송(egress) 비용이 변수.

## 5) 인증 & 보안 부가 서비스
- 외부 인증 프로바이더: 무료 구간 후 사용량 기반 과금.
- 모니터링/로깅(APM, 에러 트래커): 무료/저가 구간 존재, 초기 월 $0–$20 내로 계획 가능.

## 6) 운영 비용 통제 장치
- 레이트리밋/사용자별 쿼터로 API 비용 예측성 확보.
- AI/호스팅 월/주간 비용 알림 설정.
- 캐싱/결과 재사용으로 중복 호출 최소화.

## 7) 팀/개발 도구(선택)
- CI/CD 러너: 무료 구간이 많음, 빌드 시간 초과 시 추가 과금 가능.
- 피처 플래그/애널리틱스: 초기 무료, 이벤트/사용자 수에 따라 확장 과금.
