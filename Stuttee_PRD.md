# Stuttee (S.T) – MVP PRD

## 1. 서비스 개요
Stuttee는 사용자가 공부한 내용을 입력하면 시험 스타일의 문제, 정답, 해설을 자동 생성하는 AI 학습 보조 웹앱이다.

### 목표
- 학습 내용을 즉시 문제화하여 복습 효율을 향상.
- 자신만의 시험 대비 문제은행을 빠르게 생성.

### 핵심 가치
- 빠른 문제 생성
- 단일 페이지에서 즉시 결과 확인
- 난이도·유형 조절
- 해설 포함

---

## 2. 서비스명 & 브랜드 컨셉
### 서비스명
**Stuttee (S.T)** — Study + Tuttee(튜티)

### 브랜드 컨셉
- 가볍고 똑똑한 AI 학습 도우미
- 주요 컬러: 화이트 + 라이트 블루
- 마스코트/로고는 ‘S.T’ 심볼 기반 확장 가능

---

## 3. 타겟 사용자
- 대학생, 고등학생, 자격증 준비생
- 강의/교재 요약을 문제로 변환해 확인하고 싶은 학습자
- 시험/과제 대비 자동화 학습 루틴이 필요한 사람

---

## 4. 사용자 시나리오 (User Flow)

1) 학습 내용을 텍스트로 붙여넣는다  
2) 문제 생성 옵션 선택  
   - 문제 수 (5/10/20)  
   - 난이도 (상/중/하)  
   - 문제 유형: 객관식/서술형/혼합  
3) “문제 생성” 클릭  
4) 문제 + 정답 + 해설 출력  
5) 복사/텍스트 저장 등 간단한 내보내기 기능 활용

---

## 5. MVP 기능 스코프

### 포함 기능 (MVP)
- 텍스트 입력 박스
- 문제 생성 옵션 UI
- 문제 생성 백엔드 요청
- 문제·보기·정답·해설 표시
- 정답 토글
- 전체 복사 기능
- 기본 반응형 UI

### 제외 기능 (후순위)
- 로그인/회원 시스템
- 문제 저장/히스토리
- OCR
- 시험 모드
- 고급 통계 분석

---

## 6. 페이지 구성 (단일 페이지 SPA)

### 메인 페이지
- 상단: Stuttee 로고 + 한 줄 설명
- 입력 영역:
  - 텍스트 입력 박스
  - 문제 옵션 선택
  - [문제 생성] 버튼
- 결과 영역:
  - Q1 ~ Qn 문제 리스트
  - 객관식 보기(있을 때)
  - 정답/해설 토글
  - [전체 복사]

---

## 7. 기술 구조 (기획 수준)

### Frontend
- HTML + CSS + JS 또는 React  
- SPA 기반

### Backend
- Node.js + Express  
- POST `/generate` — OpenAI API 연동하여 문제 생성

### Database
- 없음(MVP 단계에서는 저장 기능 제외)

---

## 8. AI 프롬프트 설계 (기획 목적)

```
아래 학습 내용을 바탕으로 시험 스타일 문제를 생성하라.
- 문제 수: {count}
- 난이도: {level}
- 문제 유형: {type}
- 문제 형식은 다음 구조를 따른다:

Q1. 문제
① 보기
② 보기
정답: ...
해설: ...
```
